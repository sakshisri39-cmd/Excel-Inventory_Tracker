# Excel Inventory Tracker

**Problem:** Manual stock tracking causes delays & stockouts in warehouses.

**Solution:** Automated Excel system that flags products needing reorder.

**Key Features:**
- IF + VLOOKUP formulas for real-time stock alerts
- "REORDER NOW" auto-trigger when Current_Stock < Reorder_Level
- Pivot Tables for warehouse-wise consumption reports
- Data validation to prevent entry errors

**Skills Used:** MS Excel, Data Analysis, Inventory Management, Supply Chain Ops

**How to Use:**
1. Open `excel stock.xlsx`
2. Update 'Current_Stock' column daily
3. Check 'Status' column for alerts

**Business Impact:** Reduces manual checking by 60% + prevents stockouts
