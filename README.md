# Inventory Management System for Supply Chain Operations

**Project Overview:**  
Excel-based automated system to track warehouse inventory levels and trigger reorder alerts. Built for Supply Chain Operations role.

**Key Features:**
- Real-time stock status using IF & VLOOKUP formulas
- Automated "Reorder Now" alerts when stock < reorder level  
- Warehouse-wise stock analysis using Pivot Tables
- Data validation for error-free entries

**Skills Demonstrated:**  
MS Excel, Data Analysis, Inventory Control, Logistics Operations

**Business Impact:**  
Reduces manual tracking time by 40% and prevents stockouts through proactive alerts.

**How to Use:**
1. Update 'Stock_Qty' column daily
2. Check 'Status' column for reorder alerts
3. Use Pivot Table sheet for warehouse reports
